Project Information

Project Title: Library System

Project Languages: HTML, CSS, C#

Additional Tools Used: Blazor

Project Description:

This project was created for a university class (Human-Computer Interaction). This term-long project allowed us to design and implement a library system from scratch (from sketches to functional prototype).

The user is able to:

  • Login
  • Look up an item (book, dvd, cd, event)
  • Sign up for an event (for 1 or multiple seats)
  • Place an item on hold
  • Select pickup branch
  • Add item to wishlist
  • Remove item from wishlist
  • Renew item checked out

Project Specifics

Things I Learned:

  • Basic principles of C# (i.e. functions, variables, etc)
  • Learning the difference between the different "display" properties (i.e. flex vs. block)
  • Adding HTML elements depending on certain conditions using C#
  • How to have a handful of centralized file that will contain our user and item information that will allow other Razor pages and components access that information
      • Solution:
    • Injecting dependency
    • Adding singleton service
  • How to pass information on a specific item to a Modal in a foreach loop (rather than the modal only being passed the last item in the loop)

Things I Enjoyed:

  • Learning basic C# syntax and functionality
  • Designing the look of the website
  • Figuring out how to make the code as easy to understand (and access) as possible for other group members
  • Creating template layouts for an item page and learning how to change the details to be specific to that item (i.e. get the photo for that specific item)

Things I'd do Differently:

  • Hook up an actual database to the system
      Comment:
    • We were told not to worry about creating an SQL database, or the like, and were encouraged to hold our data in arrays/lists
    • As a result, a lot of time was spent into trying to make our "database" accessbile to all Razor pages and components
    • The solution to this, as stated above, was to create services and to inject them into the files that needed them

Challenges that were Faced:

  • Understanding how to user dependency injection
