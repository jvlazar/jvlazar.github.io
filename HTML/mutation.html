Mutation Inducer

Project Information

Project Title: Mutation Inducer

Project Languages: Python

Project Description:

The goal of this small project was to automate a mutation in an amino acid sequence given the appropriate FASTA file, amino acid location, and the action of deleting or changing the amino acid at that location.

This script was created for an Intro to Bioinformatics class, in which our group project was looking to identify whether animals with the same mutation in a specific amino acid sequence would have the same 3D structure characteristics as those in humans.

Project Specifics

Things I Learned:

  • Manipulating strings in Python
  • File handling in Python

Things I Enjoyed:

  • Error-checking to ensure that the user knows whether there is that specific amino acid at that location

Things I'd do Differently:

  • Have all the code in one file. As it is, there's a separate file to delete and to substitute
  • Add the ability to insert an amino acid in the sequence

Challenges that were Faced:

  • Getting the script to ignore the header information of the FASTA file, and to start reading at a certain location in the sequence
