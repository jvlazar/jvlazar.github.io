Project Information

Project Title: Space Tourism Multi-Page Website

Project Languages: HTML, CSS, JavaScript

Additional Tools Used: React

Project Description:

This project replicates the Space Tourism Multi-Page Website from the design images and assets provided by Frontend Mentor.

This project also had the additional goal of learning and using React Routes to help with page navigation.

Project Specifics

Things I Learned:

  • React Router
  • How to use the CSS property position: absolute to help with positioning of elements
  • How to create an image carousel
  • How to create tabs

Things I Enjoyed:

  • Using the map() function to create React Components that differ over values given in the accompanying JSON file

Things I'd do Differently:

  • Spend more time thinking about the layout of the page and not be extra relient on the position: absolute CSS property of an element
  • Add the functionality where, on mobile, if the user has the hamburger menu open and they click anywhere outside of the menu, the menu will close. Currently, user needs to click on the menu button to close it

Challenges that were Faced:

  • Positioning all the elements differently between different screen sizes
  • Positioning the hamburger menu to be on top of all content
  • Ensuring all elements look good between different screen sizes
