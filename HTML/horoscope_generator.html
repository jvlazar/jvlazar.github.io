Project Information

Project Title: Horoscope Generator

Project Languages: HTML, CSS, JavaScript

Additional Tools Used: React, Node JS

Project Description:

This project shows the daily horoscope for a given Western zodiac sign.

This project was done using React as the frontend library and Node JS/Express JS for backend code. Namely, the frontend was responsible for displaying the user's zodiac sign and the corresponding daily horoscope, while the backend was responsible for making API calls to get the horoscope.

Project Specifics

Things I Learned:

  • Basics of Node/Express JS
    • How to set up a server
    • How to receive data from the frontend
    • How to send data to the frontend
    • How to use custom Node modules
  • How to GET and POST data to/from APIs

Things I Enjoyed:

  • Learning how to connect the frontend code (a React project) with the backend code

Things I'd do Differently:

  • Pick a different type of generator that had plenty of functional (and free) APIs available

Challenges that were Faced:

  • Finding free APIs will give me the information I need
    1. Found an API gave me the daily horoscope for a given zodiac sign, but was only available in Portuguese
    2. Had to find a free API that would translate from a given language (in this case, Portuguese), to English
    3. Translator API had a limit to the amount of characters it can translate per day.
      • Resolved by making one large call to get all the zodiac signs' horoscope translated at once and save it for the day, rather than make an API call to translate every time a user selects their zodiac sign
  • Getting the data sent to the frontend automatically once it's been calculated from the backend
