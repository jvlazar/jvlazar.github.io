Project Description:

Project Specifics

Things I Learned:

  • How to use a grid layout
  • How to change the background colour of a div after a certain point (using background: linear-gradient)

Things I Enjoyed:

  • Learning how to start with a mobile-first approach, and add detail as screen size increased

Things I'd do Differently:

  • Set the background color to be solid at the start of the project, then at the end change the colour scheme to match that of the design, to ensure that no other elements' positions will change the screen size and as a result change the %s for the linear gradient

Challenges that were Faced:

  • Getting the elements on the screen to be responsive
  • Learning how to change the background color midway through a div (using the background: linear-gradient property solved this challenge)
