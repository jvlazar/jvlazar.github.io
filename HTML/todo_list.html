Project Information

Project Title: To-Do List

Project Languages: HTML, CSS, JavaScript (React)

Additional Tools Used: React

Project Description:

This project replicates a to-do list in which the user can create new tasks, delete existing tasks, and complete in-progress tasks. The user can also view their completed tasks and add them back into the in-progress tab or delete them. Tasks are ordered chronologically, with the most recently created task showing at the top of the list.
